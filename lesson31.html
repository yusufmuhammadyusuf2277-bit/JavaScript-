Setting Up Your JavaScript Project

Before starting a project, it is important to organize your files, set up HTML, CSS, and JavaScript correctly, and structure folders for future scalability. 

index.html
style.css
script.js

Include the script and style in your HTML: 

<link rel="stylesheet" href="style.css">
<script src="script.js"></script>
